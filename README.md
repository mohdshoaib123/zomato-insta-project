# FoodReels - Instagram × Zomato Clone

A full-stack social food discovery platform inspired by **Instagram Reels** and **Zomato**, where users can explore engaging food-related short videos, discover trending dishes, and interact with food content. The platform includes secure user authentication and an admin dashboard for managing and uploading food reels.

## 🚀 Features

### User Features

* User Registration & Login
* Secure Authentication
* Browse Food Reels
* Infinite Reel Scrolling Experience
* Mobile-Friendly Responsive UI
* Food Content Discovery

### Admin Features

* Admin Authentication
* Upload Food Reels
* Manage Reel Content
* Control Platform Content

## 🏗️ Project Overview

FoodReels combines the engaging short-video experience of Instagram with the food discovery concept of Zomato. Users can watch food-related reels uploaded by administrators, making it easier to discover trending foods, restaurants, and recipes through interactive video content.
